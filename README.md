源码+数据库+全套环境和工具资源+本地部署教程

### 五、项目介绍
项目编号：mpweixinA209D

房屋租赁租借微信小程序为房东和管理者提供高效的房源与租客管理服务，实现租赁业务的数字化与智能化。

角色：管理员、用户、房东

管理员：首页、系统管理、用户管理、房源信息管理、预订信息、电子合同管理。

用户：首页、房源信息、我的、预订信息、电子合同、我的收藏、聊天记录、修改密码。

房东：首页、房源信息管理、预订信息、电子合同管理。
